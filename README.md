# Blood Bank Management System

This repository contains a full-stack Blood Bank Management System with a modern UI and robust backend.

## 📂 Repository Structure

- [**`backend/`**](./backend) - Spring Boot source code and Maven setup.
- [**`frontend/`**](./frontend) - UI Templates (Thymeleaf) and Styling.
- [**`api/`**](./api) - API Documentation and Endpoints.
- [**`documentation/`**](./documentation) - Detailed project guide and setup instructions.

## 🚀 Quick Start
To run the project, go to the `backend/` folder and use the included Maven wrapper to run:
`mvn spring-boot:run`

Refer to [**Documentation**](./documentation/Project_Guide.md) for detailed steps.
